### Solution
```python
def sockMerchant(n, ar):
    f = {}
    count = 0
    for i in ar:
        if i in f:
            f[i] += 1
        else :
            f[i] = 1
        if f[i] == 2:
            count += 1
            f[i] = 0
    return count
```
